02 Partition Design

Two partitions. EFI is unencrypted (bootloader reads it before LUKS opens). Everything else — including the swap file — lives inside the LUKS container. One passphrase unlocks the entire system. TPM2 can automate that after enrollment.

EFI 1G
LUKS2 (rest of disk — all data inside)
@
@home
@nix
@log
@cache
@tmp
@snap
@persist
@swap
p1 — EFI — 1G — FAT32 — unencrypted
p2 — LUKS2 → BTRFS with 9 subvolumes (including swap)

BTRFS Subvolume Map

LUKS2 → cryptroot (/dev/mapper/cryptroot)
└─ BTRFS "nixos"
   ├─ @ → / root filesystem
   ├─ @home → /home user data
   ├─ @nix → /nix nix store — lighter compression
   ├─ @log → /var/log nodatacow
   ├─ @cache → /var/cache nodatacow
   ├─ @tmp → /tmp nodatacow
   ├─ @snapshots → /.snapshots snapper target
   ├─ @persist → /persist impermanence-ready
   └─ @swap → /swap nodatacow, no compress — swapfile lives here
Why swapfile inside BTRFS instead of a separate partition? A separate swap partition would be outside LUKS (or need its own LUKS device + key management). A swapfile inside @swap is entirely inside the LUKS container — swap data is always encrypted. It also supports hibernation without a second passphrase prompt. The only overhead is that BTRFS swapfiles require nodatacow and no compression on the subvolume.
03 disko.nix — The Disk Declaration

Already in your repo at nixos-config/disko.nix. Run once during install. Creates partitions, formats LUKS+BTRFS, creates all subvolumes, mounts to /mnt. You create the swapfile separately in step 5.

nixos-config/disko.nix — key partsnix
{ ... }:
{
  disko.devices.disk.main = {
    type   = "disk";
    device = "/dev/nvme0n1";    # ← confirm with lsblk
    content = {
      type = "gpt";
      partitions = {
        ESP = {
          end  = "1GiB"; type = "EF00";
          content = { type = "filesystem"; format = "vfat"; mountpoint = "/boot"; };
        };
        luks = {
          size = "100%";
          content = {
            type = "luks";
            name = "cryptroot";
            extraOpenArgs = [ "--allow-discards" "--perf-no_read_workqueue" "--perf-no_write_workqueue" ];
            settings.allowDiscards = true;
            content = {
              type = "btrfs"; extraArgs = [ "-L" "nixos" "-f" ];
              subvolumes = {
                "@"          = { mountpoint = "/";           mountOptions = [ "noatime" "compress=zstd:3" ... ]; };
                "@home"      = { mountpoint = "/home";        mountOptions = [ "noatime" "compress=zstd:3" ... ]; };
                "@nix"       = { mountpoint = "/nix";         mountOptions = [ "noatime" "compress=zstd:1" ... ]; };
                "@log"       = { mountpoint = "/var/log";     mountOptions = [ "noatime" "nodatacow"        ... ]; };
                "@cache"     = { mountpoint = "/var/cache";   mountOptions = [ "noatime" "nodatacow"        ... ]; };
                "@tmp"       = { mountpoint = "/tmp";         mountOptions = [ "noatime" "nodatacow"        ... ]; };
                "@snapshots" = { mountpoint = "/.snapshots";  mountOptions = [ "noatime" "compress=zstd:3" ... ]; };
                "@persist"   = { mountpoint = "/persist";     mountOptions = [ "noatime" "compress=zstd:3" ... ]; };

                # Swap subvolume — nodatacow, NO compression (required for BTRFS swapfiles)
                "@swap"      = { mountpoint = "/swap";        mountOptions = [ "noatime" "nodatacow"        ... ]; };
              };
            };
          };
        };
      };
    };
  };
}
04 NixOS Config Changes

All already done in your repo. Reviewing what each file does.

modules/nixos/disks.nix — LUKS + filesystems + swapfile + hibernation

modules/nixos/disks.nixnix
{ config, lib, ... }:
{
  boot.initrd.luks.devices."cryptroot" = {
    device           = "/dev/disk/by-partlabel/luks";
    allowDiscards    = true;    # TRIM through LUKS
    bypassWorkqueues = true;    # reduce NVMe latency
  };

  boot.initrd.supportedFilesystems = [ "btrfs" ];
  boot.supportedFilesystems        = [ "btrfs" ];

  fileSystems = let
    btrfsOpts = [ "noatime" "compress=zstd:3" "space_cache=v2" "ssd" "discard=async" ];
    nixOpts   = [ "noatime" "compress=zstd:1" "space_cache=v2" "ssd" "discard=async" ];
    cowOpts   = [ "noatime" "nodatacow"        "space_cache=v2" "ssd" "discard=async" ];
    dev       = "/dev/mapper/cryptroot";
  in {
    "/"           = { device = dev; fsType = "btrfs"; options = [ "subvol=@"          ] ++ btrfsOpts; };
    "/home"       = { device = dev; fsType = "btrfs"; options = [ "subvol=@home"      ] ++ btrfsOpts; };
    "/nix"        = { device = dev; fsType = "btrfs"; options = [ "subvol=@nix"       ] ++ nixOpts; };
    "/var/log"    = { device = dev; fsType = "btrfs"; options = [ "subvol=@log"       ] ++ cowOpts; neededForBoot = true; };
    "/var/cache"  = { device = dev; fsType = "btrfs"; options = [ "subvol=@cache"     ] ++ cowOpts; };
    "/tmp"        = { device = dev; fsType = "btrfs"; options = [ "subvol=@tmp"       ] ++ cowOpts; };
    "/.snapshots" = { device = dev; fsType = "btrfs"; options = [ "subvol=@snapshots" ] ++ btrfsOpts; };
    "/persist"    = { device = dev; fsType = "btrfs"; options = [ "subvol=@persist"   ] ++ btrfsOpts; neededForBoot = true; };
    "/swap"       = { device = dev; fsType = "btrfs"; options = [ "subvol=@swap"      ] ++ cowOpts; };
    "/boot"       = { device = "/dev/disk/by-partlabel/ESP"; fsType = "vfat"; options = [ "fmask=0022" "dmask=0022" ]; };
  };

  swapDevices = [{ device = "/swap/swapfile"; }];

  # Hibernation — resume from the encrypted BTRFS swapfile.
  # resume_offset: get it during install (step 5):
  #   sudo btrfs inspect-internal map-swapfile -r /mnt/swap/swapfile
  # Replace 0 with the "physical start" value, then rebuild.
  boot.resumeDevice = "/dev/mapper/cryptroot";
  boot.kernelParams = [ "resume_offset=0" ];    # ← replace with actual offset
}
resume_offset must be set during install. You cannot compute it before creating the swapfile. Step 5 walks through this. After getting the value, edit disks.nix, replace 0, and proceed with nixos-install.

modules/nixos/tpm.nix — NEW (import after first boot)

TPM2 auto-unlock. Do not import this yet — import it after first successful passphrase boot and after running enrollment (step 7).

modules/nixos/tpm.nix — import after enrollmentnix
{ config, lib, pkgs, ... }:
{
  # systemd initrd: required for TPM2 LUKS unlock
  boot.initrd.systemd.enable = true;

  # Tell systemd's crypttab to try TPM2, fall back to passphrase
  boot.initrd.luks.devices."cryptroot".crypttabExtraOpts = [
    "tpm2-device=auto"
    "tpm2-pcrs=0+2+7+15"    # see step 7 for PCR explanation
  ];

  security.tpm2 = {
    enable                 = true;
    pkcs11.enable          = true;
    tctiEnvironment.enable = true;
  };

  environment.systemPackages = with pkgs; [ tpm2-tools ];
}

3 — Regenerate hardware-configuration.nix

required on any hardware — replaces the original machine's configbash
# Generate from the mounted system (--no-filesystems: disks.nix owns those)

# Copy the generated file into the cloned repo

   /tmp/nixos-config/hosts/nixos/hardware-configuration.nix
Do not skip this on new hardware. The repo ships with hardware-configuration.nix from the original machine. New hardware may have different kernel modules (storage controller, CPU microcode, GPU). Skipping this means the initrd may not find the NVMe drive — you get a black screen after GRUB with no LUKS prompt.

4 — Create the swapfile

BTRFS swapfile — must use btrfs mkswapfile, not fallocate/ddbash
# Adjust size to your RAM (for full hibernation, size >= RAM)
sudo btrfs filesystem mkswapfile --size 16G /mnt/swap/swapfile

# Verify it was created correctly
ls -lh /mnt/swap/swapfile

5 — Get resume_offset and update config

compute hibernation resume offset — do this nowbash
sudo btrfs inspect-internal map-swapfile -r /mnt/swap/swapfile

# Output looks like:
# Swapfile /mnt/swap/swapfile:
#         UUID:            xxxxxxxx-xxxx-...
#         physical start:  4096000        ← copy THIS number
#         flags:           0x1

# Edit disks.nix — uncomment the two hibernation lines and replace the placeholder
nano /tmp/nixos-config/modules/nixos/disks.nix
# Uncomment: boot.resumeDevice = "/dev/mapper/cryptroot";
# Uncomment: boot.kernelParams = [ "resume_offset=4096000" ];
resume_offset must match the swapfile. If you ever recreate or resize the swapfile (e.g. with btrfs filesystem mkswapfile --size 32G), recompute and update this value, then nixos-rebuild switch.

07 TPM2 Auto-Unlock

After a successful passphrase boot, you can enroll the TPM chip to auto-unlock LUKS. The TPM seals the LUKS key against PCR measurements — if the firmware or Secure Boot state changes unexpectedly, the TPM refuses to release the key and falls back to asking for your passphrase.

PCR policy — what each register measures

PCRMeasuresBreaks TPM if...Use?
0UEFI firmware code Firmware update ✓ Yes
2Option ROMs / external firmware eGPU, new Thunderbolt device ✓ Yes
7Secure Boot state + keys SB enabled/disabled, key change ✓ Yes
12Kernel command line Any NixOS rebuild that changes kernel params ⚠ Optional
15systemd OS phase marker Major systemd version changes ✓ Yes
Recommended: PCR 0+2+7+15. This survives NixOS generation switches and nixos-rebuild switch without re-enrollment. PCR 12 adds extra security but requires re-enrollment whenever kernel params change (which can happen across rebuilds). Add it only if you want maximum security and are willing to re-enroll after certain rebuilds.

08 zram — In-Memory Compressed Swap

Three-layer memory stack: RAM → zram (compressed, in-memory, fast) → swapfile (encrypted on disk, fallback + hibernation). zram is 10–40× faster than the swapfile and kicks in long before the disk is ever touched.

add to nix-settings.nix or any modulenix
zramSwap = {
  enable        = true;
  algorithm     = "zstd";
  memoryPercent = 50;    # 50% of RAM = zram device size
};

boot.kernel.sysctl = {
  # 180: push pages to zram aggressively (safe — zram is in-memory)
  "vm.swappiness"             = 180;
  "vm.page-cluster"           = 0;     # swap one page at a time (good for zram)
  "vm.vfs_cache_pressure"     = 50;
  "vm.dirty_ratio"            = 6;
  "vm.dirty_background_ratio" = 2;
};
Priority order: zram gets priority 100 (runs first). The swapfile gets priority -2 (disk fallback). The kernel will fill zram almost completely before any swap hits the (encrypted) disk. Hibernation writes RAM to the swapfile on disk (also encrypted via LUKS).

10 Full Verification Checklist
run all checks after first bootbash
## ── 1. LUKS ─────────────────────────────────────────────────
cryptsetup status cryptroot
# type: LUKS2 | cipher: aes-xts-plain64 | keysize: 512 bits

## ── 2. Partitions ────────────────────────────────────────────
ls -la /dev/disk/by-partlabel/
# ESP → ../nvme0n1p1
# luks → ../nvme0n1p2

## ── 3. BTRFS subvolumes ──────────────────────────────────────
sudo btrfs subvolume list /
# Expected: @, @home, @nix, @log, @cache, @tmp, @snapshots, @persist, @swap

## ── 4. Mount options ─────────────────────────────────────────
findmnt -t btrfs -o TARGET,OPTIONS
# / should show compress=zstd:3, noatime, ssd, discard=async
# /var/log /var/cache /tmp /swap should show nodatacow

## ── 5. Swap stack ────────────────────────────────────────────
swapon --show
# NAME           TYPE  SIZE  USED PRIO
# /dev/zram0     ...    8G    0B  100   ← zram (primary)
# /swap/swapfile file  16G    0B   -2   ← swapfile (fallback + hibernate)

## ── 6. Hibernation ───────────────────────────────────────────
cat /proc/cmdline | tr ' ' '\n' | grep resume

# resume_offset=XXXXXXX   ← should NOT be 0 if you set it correctly

## ── 7. TRIM through LUKS ─────────────────────────────────────
sudo fstrim -v /        # should succeed

## ── 8. TPM status (after step 7) ─────────────────────────────
